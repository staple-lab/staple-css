# Gradient Tokens

Pre-defined, perceptually-smooth gradients for modern, accessible design. Use with the background or background-image CSS properties.

---

## How to Use

Apply gradients via CSS variables:

```css
.hero {
  background: var(--st-gradient-sunrise);
  /* or */
  background-image: var(--st-gradient-ocean);
}
```

---

## Tone-Based Gradients

Systematic gradients for each tone (neutral, primary, danger, warning, success) in three intensities: soft, medium, and bold.

### Neutral Gradients
- **Neutral Soft**: Light neutral gradient for subtle backgrounds
- **Neutral Medium**: Medium neutral gradient for balanced contrast
- **Neutral Bold**: Dark neutral gradient for emphasis

### Primary Gradients
- **Primary Soft**: Light blue gradient for primary accents
- **Primary Medium**: Medium blue gradient for brand colors
- **Primary Bold**: Dark blue gradient for strong actions

### Danger Gradients
- **Danger Soft**: Light red gradient for warning states
- **Danger Medium**: Medium red gradient for destructive actions
- **Danger Bold**: Dark red gradient for critical alerts

### Warning Gradients
- **Warning Soft**: Light orange gradient for cautions
- **Warning Medium**: Medium orange gradient for warnings
- **Warning Bold**: Dark orange gradient for emphasis

### Success Gradients
- **Success Soft**: Light green gradient for confirmation
- **Success Medium**: Medium green gradient for success states
- **Success Bold**: Dark green gradient for positive feedback
