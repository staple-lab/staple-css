## Best Practices

### Do
- Use token scales: pad={0-8}
- Compose with Box & Flex
- Use semantic token keys
- Leverage type safety

### Don't
- Use raw pixel values
- Create custom paddings
- Hardcode colors
- Ignore TypeScript warnings
